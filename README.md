📝 To-Do List App

Welcome to my very first Web Development project! 🚀

📖 About

This is a simple, elegant, and fully functional To-Do List application built using core web technologies. It allows users to add, complete, and manage their daily tasks. The best part? It uses the browser's LocalStorage, so your tasks are never lost even if you refresh the page or close the browser!

This project marks my official entry into the world of Web Development and my first ever push to GitHub.

✨ Features

Add Tasks: Type a task and hit Enter or click the "Add Task" button.

Mark as Complete: Click on any task to cross it out.

Delete Tasks: Remove individual tasks using the delete button.

Clear All: A dedicated "Clear All" button to wipe out all tasks at once.

Data Persistence: Uses LocalStorage to save tasks permanently.

Dark UI: A clean, modern dark-themed user interface with smooth hover animations.

🛠️ Tech Stack

HTML5: For the structural foundation.

CSS3: For styling, Flexbox layout, and responsive design.

Vanilla JavaScript (ES6+): For DOM manipulation, event handling, and logical operations.

🧠 Key Learnings

While building this project, I practically implemented several core software development concepts:

DOM Manipulation: Dynamically creating elements (document.createElement) and updating the UI without reloading.

Event Handling: Capturing mouse clicks and keyboard events (keydown).

Event Bubbling: Learned how to isolate click events using e.stopPropagation().

State Management: Using JSON.stringify() and JSON.parse() to sync a JavaScript Array with browser LocalStorage.

Version Control: Setting up Git, generating SSH keys, and pushing code to a remote GitHub repository.
